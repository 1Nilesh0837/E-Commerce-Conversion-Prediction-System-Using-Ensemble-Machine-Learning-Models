Key EDA Findings:

  • Users with higher Pages_Viewed and Products_Viewed showed significantly higher conversion rates.
  • Previous_Purchases was positively correlated with conversion.
  • Campaign_Code exhibited large variation in conversion rates across ~9,000 unique values and was therefore selected for target encoding.

2. Data Cleaning

  • Missing Indicators: Created binary flags (Age_missing, Income_missing, Time_missing, total_missing) — missingness itself can be predictive
  • Imputation: Filled missing values with median (Age=41.0, Income=70,171.6, Time_On_Site=11.1) — computed only on training data to prevent leakage
  • Outlier Handling: Time_On_Site had extreme outliers (max=607.4 vs median=11.1) — created a clipped version capped at 60
  • Data Augmentation: After model selection and validation, train and public_test labels were combined to retrain the final ensemble before generating private_test predictions.

5. Best Model: Weighted Ensemble

🏆 Best Model = Weighted Ensemble of CatBoost (60%) + LightGBM (15%) + XGBoost (25%)
Optimal threshold: 0.26  |  Ensemble CV F1: 0.56706

The ensemble was selected because it consistently outperformed individual models during cross-validation while reducing model-specific variance.

Key Techniques That Boosted Performance:

  • Threshold Optimization: Instead of default 0.5, searched [0.25, 0.65] to find the optimal decision boundary, which improved validation F1 score compared to the default threshold.
  • Ensemble Weight Search: Grid search over CatBoost, LightGBM, XGBoost weight combinations
  • Train+Public Retraining: Retraining on combined train and public test labels to maximize data exposure before final predictions.
  • Feature Engineering: 20 domain-specific features capturing engagement, intent, financial behavior
  • Leak-free Encoding: KFold target encoding prevents overfitting on high-cardinality features
